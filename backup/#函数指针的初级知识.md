##函数指针的每个写的东西对应的意思
返回值类型 (*指针变量名)(参数类型列表);
例子:        ```c
int (*func_ptr)(int, int);
```
运用的例子：
```c
#include <stdio.h>

int add(int a, int b) {
return a + b;
}

int subtract(int a, int b) {
return a - b;
}

int main() {
int (*operation)(int, int);
operation = add;
    printf("3 + 5 = %d\n", operation(3, 5)); 
    operation = subtract;
    printf("10 - 4 = %d\n", operation(10, 4));
    return 0;
}
```
用typedef的简易的方法的例子
```c
#include <stdio.h>
typedef int (*MathFunc)(int, int);
int multiply(int a, int b) {
    return a * b;
}
int main() {
    MathFunc func;  
    func = multiply;
    printf("3 * 4 = %d\n", func(3, 4)); 
    return 0;
}```
